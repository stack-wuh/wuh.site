## 启动项目
1. master分支，有create-react-app创建的react项目
2. v2分支为Nextjs构建的react项目

```bash
git clone -b v2 https://github.com/stack-wuh/react-router-config.git next-blog

cd next-blog
yarn

yarn run dev -p 5544
```
